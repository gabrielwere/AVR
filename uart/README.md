### UART
Send the word 'Hello' to a device connected via uart.<br/>
HyperTerminal is used to view the data in place of a uart connected device<br/>

### HyperTerminal settings
1. Baud Rate - 9600<br/>
2. Stop bits - 1<br/>
3. Parity bit - None<br/>
4. Data bits - 8<br/>
5. Flow Control - None<br/>
